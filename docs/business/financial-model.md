## 11. Оборотный капитал (Working Capital)

- Правило стока: **60 дней покрытия** → `Inventory = 2 × COGS следующего месяца`.
- Отсрочка поставщиков: 0 дней (M1–6) → 30 дней (с M7) → `Payables = 1 × месячный COGS`.
- Net WC = Inventory − Payables ≈ 1 месяц COGS: **€22k (конец Y1) → €45k (Y2) → €105k (Y3)**.
- ΔWC по годам: −22 / −22,5 / −59,5 k€ (потребление кэша ростом).

## 14. Точка безубыточности

- Формула: `BE-заказы/мес = Fixed_мес / (Contribution/заказ − Маркетинг/заказ)`.
- На структуре Y3: 27 000 / (14,5 − 5,8) = **~3 100 заказов/мес ≈ €152k/мес выручки** → достигается **M31–32** (см. §6–7).

## 17. Формулы для воспроизведения (Excel/Sheets)

```
NewCustomers[m]   = Marketing[m] / CAC[m]
CustBase[m]       = CustBase[m-1] + NewCustomers[m]
RepeatOrders[m]   = CustBase[m-1] * RepeatRate[m]
Orders[m]         = NewCustomers[m] + RepeatOrders[m]
Revenue[m]        = Orders[m] * AOV[m]                     // нетто, без НДС
COGS[m]           = Revenue[m] * (1 - GM%[m])
Logistics[m]      = Orders[m] * NetShipCost[m]
PaymentFees[m]    = Revenue[m]*1.8% + Orders[m]*0.25
Returns[m]        = Revenue[m] * 2%
Contribution[m]   = Revenue - COGS - Logistics - PaymentFees - Returns
EBITDA[m]         = Contribution[m] - Marketing[m] - Fixed[m]
Inventory[m]      = 2 * COGS[m+1]
Payables[m]       = 1 * COGS[m]          // с M7
dWC[m]            = (Inventory[m]-Payables[m]) - (Inventory[m-1]-Payables[m-1])
Cash[m]           = Cash[m-1] + EBITDA[m] - dWC[m] - OneOff[m] + Financing[m]
LTV               = OrdersPerCustomer24m * ContributionPerOrder
BE_orders         = FixedMonthly / (ContributionPerOrder - MarketingPerOrder)
StaffCost         = Gross * 1.2359       // соцвзносы работодателя, Латвия
```

Проверки целостности (добавить в лист): `Σмесяцы = год`, `CAC_blended = ΣMarketing/ΣNew`, `маржа EBITDA монотонно растёт с Q5`, `Cash[m] > 0 ∀m`.

## Резюме CFO

1. Модель подтверждает тезис бизнес-плана: **экономика живёт на повторных заказах** — LTV/CAC растёт с 1,5 (ранние когорты) до 3,4 (Y3) исключительно за счёт repeat rate; чувствительность к CAC вторична.
2. **Честная поправка к отчёту 08:** €200k хватает на 18 месяцев, но не на 36. Полный план требует €350k (bridge €150k на M18–20, решение по данным M15–16). Альтернатива — оптимистичная траектория или сдвиг LT-запуска на полгода (экономит ~€60k, но откладывает масштаб).
3. Самая опасная точка — **M18: кэш ≈ €9k** при одновременном запуске Литвы. Митигации: отсрочки поставщиков (45 дней = +€35k кэша), гейт LT-запуска, ранний старт разговора о bridge (M12).
4. Два операционных рычага важнее всех остальных: **логистика на заказ** (±€1 = ±400 заказов к точке безубыточности) и **repeat rate** (+1 п.п. месячного rate ≈ +€100k выручки Y3 почти без затрат).
